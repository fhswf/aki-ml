# Applied Artifical Intelligence - Machine Learning

This repository containes everything for the module *Machine Learning* in the Applied Artificial Intelligence Program from South Westphalia University of Applied Sciences. 

# Installation

To install the required packages, run the following command in your terminal:

```bash
py -m venv .venv

.venv/bin/activate  # On Linux/ macOS, use `source .venv\Scripts\activate

pip install -r requirements.txt
```

# Development

## Filter Jupyter Notebook

The Jupyter Notebooks save their output and other metadata. By using **nb-clean** the jupyter notebook is stripped of this output and metadata.

```bash
nb-clean clean path_to_file # Manual cleaning
```
